PyGNSS-TEC is a Python package with Rust-accelerated performance, designed for processing and analyzing Total Electron Content (TEC) data derived from Global Navigation Satellite System (GNSS) observations. The package provides tools for RINEX file reading, TEC calculation, and DCB correction to facilitate ionospheric studies.
